## BagoShop

1. Levantar la Base de datos
```
docker-compose up -d
```

2. Instalar modulos
```bash
$ npm install
```

3. Arrancar el proyecto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
